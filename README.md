# VibeChecker
Common project repository for the winter project r/VibeChecker. Fork this repository and submit your work there, I will merge the pull requests timely.

## How to submit?
**1.** Create a fork of this repo in your own account and push your changes to your FORK.
**2.** Upload your files to the week's respective folder and open a pull request.
**3.** I will merge your pull request to the main branch.
